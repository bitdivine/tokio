# Tokio - Minimal Landing Page

## Overview
A minimal landing page application that displays only the word "tokio" centered on the page.

## Features
- Display the text "tokio" centered both vertically and horizontally on the page
- Clean, minimal design with white background and black text
- Responsive layout that works on all screen sizes

## Frontend Requirements
- Single page application with minimal styling
- Text should be perfectly centered in the viewport
- Use simple, clean typography
- White background with black text
- No navigation, headers, footers, or additional content

## Backend Requirements
- No backend functionality required for this minimal landing page
- Static content only

## User Interface
- The page displays only the word "tokio" as the main content
- Text is centered both horizontally and vertically in the browser window
- Clean, minimalist aesthetic with no additional visual elements
